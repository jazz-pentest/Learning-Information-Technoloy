## What is it's purpose?
SSH is an encryption and authentication protocol
## It's Services
- Command line access
  - Servers
- Remote Access
  - Port 22
## It Supports
- Tunneling
- Secure Transfer
## It's Security
- Authentication
- Encryption
- Integrity
## Protects Against 
- Session hijacking
- Credential theft
- Data Sniffing
## Lesson Learned 
- SSH is not used in passkey
- It's not soley used to remote into things
- SSH is not a protocol for RDP
- Admins can use ssh and rdp, users can use RDP but need permission to use SSH
